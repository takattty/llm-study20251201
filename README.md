## 目次（全体: 180分 = 3時間）

### 1. イントロダクション（15分）
- 勉強会の目的と背景（5分）
- 環境設定（10分）

### 2. 知識 ～前半: LLMの基礎～（15分）
- **LLMの基礎**
  - いろんなLLM
  - LLMの仕組み
  - Instruction Tuning
  - Reasoning / Thinking
- **プロンプトエンジニアリング**
  - 基本テクニック
  - プロンプトの洗練方法
  - 良い例と悪い例の比較

### 3. 実践演習（ハンズオン）～前半～（45分 + 12分解説）
- **基礎（例題A）**（15分+4分解説）
- **構造化出力（例題B）**（15分+4分解説）
- **複数LLMに分ける（例題C）**（15分+4分解説）

### 4. 知識 ～後半： フィードバックループを持つプロダクトのために～（15分）
- **LLM時代のデータ活用**
  - プロダクト全体の最適化
    - ファインチューニング（FT）
    - 強化学習（RL）
  - パーソナライズ
    - コンテキストエンジニアリング
- **コンテキストエンジニアリング**
  - RAG
  - クエリ拡張、ハイブリッド検索、リランキング、グラウンディング
  - チャンキング
- **エージェント**
  - ReAct Agent
  - Reflexion

### 5. 実践演習（ハンズオン）～後半～（45分 + 12分解説）
- **APIオプション（例題D）**: マルチモーダル入力、コード実行、グラウンディング、Tool Calling（15分+4分解説）
- **周辺技術（例題E）**: Embedding、DeepResearch設計、LangChainのReAct Agent（15分+4分解説）
- **その他（例題F）**: n8n、トレース（15分+4分解説）

### 6. 案件の実例紹介（15分）

### 7. 締め（2分）
