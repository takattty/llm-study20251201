# Python入門：プログラミングの第一歩

Pythonは、その読みやすさと多様な用途から、世界中で非常に人気のあるプログラミング言語です。Web開発、データ分析、AI、自動化など、幅広い分野で活用されています。この入門記事では、Pythonプログラミングの基本的な概念を分かりやすく解説します。

## 1. 変数：値を格納する箱

変数は、データ（数値、文字列など）を一時的に保存するための名前付きの場所です。

```python
# 数値を代入
x = 10
# 文字列を代入
name = "太郎"
# 変数の値を出力
print(x)    # 出力: 10
print(name) # 出力: 太郎
```

## 2. リスト：複数の値をまとめる

リストは、複数の値を順序付けて格納できるデータ構造です。異なる型の値を混在させることも可能です。

```python
# 数値のリスト
numbers = [1, 2, 3, 4, 5]
# リストの要素はインデックス（0から始まる番号）でアクセス
print(numbers[0]) # 出力: 1 (最初の要素)
print(numbers[2]) # 出力: 3
```

## 3. 制御フロー：プログラムの流れを操る

### 3.1. for文：繰り返し処理

`for`文を使うと、指定した回数だけ処理を繰り返したり、リストなどのコレクションの各要素に対して処理を行ったりできます。

```python
# 0から4までを繰り返す
for i in range(5):
    print(i)

# リストの要素を一つずつ取り出して処理
fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
    print(fruit)
```

### 3.2. if文：条件分岐

`if`文は、特定の条件が真（True）の場合にのみ、コードブロックを実行します。`elif`（else if）や`else`と組み合わせることで、複数の条件に応じた処理を記述できます。

```python
x = 10

if x > 15:
    print("xは15より大きい")
elif x > 5: # 上の条件が偽で、この条件が真の場合
    print("xは5より大きい（15以下）")
else: # どの条件も真でない場合
    print("xは5以下")
# この例では「xは5より大きい（15以下）」が出力されます
```

## 4. 関数：処理をまとめる

関数は、特定のタスクを実行するコードのまとまりです。同じ処理を何度も使う場合に便利で、コードの再利用性を高めます。

```python
# 'name'という引数を受け取る関数
def greet(name):
    print("こんにちは、" + name + "さん！")

# 関数を呼び出す
greet("太郎") # 出力: こんにちは、太郎さん！

# 戻り値を持つ関数
def add(a, b):
    return a + b

result = add(3, 5)
print(result) # 出力: 8
```

## 5. 辞書：キーと値のペアで管理

辞書は、キー（鍵）と値（データ）のペアでデータを管理するデータ構造です。キーを使って値に素早くアクセスできます。

```python
# 'name'と'age'をキーとする辞書
person = {'name': '太郎', 'age': 25}
# キーを指定して値にアクセス
print(person['name']) # 出力: 太郎
print(person['age'])  # 出力: 25
```

## 6. クラス：オブジェクト指向プログラミングの基本

クラスは、オブジェクト（モノ）を作成するための設計図です。データ（属性）と、そのデータを操作する関数（メソッド）を一つにまとめます。

```python
class Dog:
    # オブジェクトが作成されるときに呼ばれる初期化メソッド
    def __init__(self, name):
        self.name = name # 'self.name'でオブジェクトの属性を設定
    
    # メソッド（オブジェクトの振る舞い）
    def bark(self):
        print(self.name + "がワン！と吠えました。")

# Dogクラスから'dog'というオブジェクトを作成
dog = Dog("ポチ")
# オブジェクトのメソッドを呼び出す
dog.bark() # 出力: ポチがワン！と吠えました。
```

## 7. エラー処理：予期せぬ事態に備える

プログラムの実行中に発生するエラー（例外）を適切に処理することで、プログラムが予期せず停止するのを防ぎ、より堅牢なアプリケーションを作成できます。`try-except`ブロックを使用します。

```python
try:
    # エラーが発生する可能性のあるコード
    result = 10 / 0
except ZeroDivisionError: # 特定のエラーを捕捉
    print("エラー：0で割ることはできません。")
except Exception as e: # その他の全てのエラーを捕捉
    print(f"予期せぬエラーが発生しました: {e}")
# この例では「エラー：0で割ることはできません。」が出力されます
```

## 8. ファイル操作：外部データとの連携

Pythonは、テキストファイルやCSVファイルなどの外部ファイルを簡単に読み書きできます。`with open(...)`構文を使うと、ファイルの閉じ忘れを防ぎ、安全に操作できます。

### 8.1. ファイルの読み込み

ファイルを読み込むには、`'r'`モードで開きます。

```python
# data.txtというファイルが存在すると仮定
# with文を使うと、ブロックを抜けるときに自動的にファイルが閉じられます
with open('data.txt', 'r', encoding='utf-8') as f:
    content = f.read() # ファイル全体を読み込む
    print("ファイルの内容：")
    print(content)

# 1行ずつ読み込む場合
with open('large_file.txt', 'r', encoding='utf-8') as f:
    print("ファイルの内容（1行ずつ）：")
    for line in f:
        print(line.strip()) # strip()で改行文字を除去
```

### 8.2. ファイルへの書き込み

ファイルを書き込むには、`'w'`（上書き）または`'a'`（追記）モードで開きます。

```python
# output.txtに書き込む（ファイルが存在しない場合は新規作成、存在する場合は上書き）
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Hello, Python World!\n')
    f.write('これは新しい行です。\n')

# existing_file.txtに追記する
with open('existing_file.txt', 'a', encoding='utf-8') as f:
    f.write('これは追記された行です。\n')
```

### 8.3. CSVファイルの読み込み

`csv`モジュールを使うと、CSVファイルを構造化されたデータとして簡単に扱えます。

```python
import csv

# data.csvというファイルが存在すると仮定
with open('data.csv', 'r', encoding='utf-8') as f:
    reader = csv.reader(f) # CSVリーダーを作成
    print("CSVファイルの内容：")
    for row in reader:
        print(row) # 各行がリストとして読み込まれる
```
